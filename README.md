📊 HR Attrition Analytics Dashboard

End-to-End Data Analytics Capstone Project

📌 Project Overview

The HR Attrition Analytics Dashboard is a full-stack data analytics project designed to analyze employee attrition patterns, workforce performance, and satisfaction trends using Python, SQL, and Power BI.

This project demonstrates a complete data pipeline:

Raw Data → Cleaning → Transformation → Database → Analysis → Dashboard → Business Insights


🎯 Business Objective

To help HR teams and management:

Identify key factors causing attrition

Monitor employee performance trends

Analyze satisfaction levels

Track early-stage attrition

Improve retention strategies

Support data-driven HR decisions

 📊 Dashboard KPIs

 🧮 Total Employees  
**Definition:** Total number of active employees  
**Business Use:** Workforce planning, capacity analysis  

 🧮 Attrition Count  
**Definition:** Number of employees who left the organization  
**Business Use:** Turnover tracking, retention analysis  

 🧮 Attrition Rate (%)  
**Formula:** Attrition Rate = Attrition Count / Total Employee
**Business Use:** HR strategy, policy optimization  

 🧮 Average Salary  
**Definition:** Mean salary of employees  
**Business Use:** Compensation planning, HR budgeting  

 🧮 Early Attrition (< 3 Years)  
**Definition:** Employees leaving within first 3 years  
**Business Use:** Onboarding effectiveness, training strategy  

 🧮 Gender Attrition Rate  
**Definition:** Attrition by gender category  
**Business Use:** Diversity & inclusion analytics
